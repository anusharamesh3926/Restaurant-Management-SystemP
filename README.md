# Restaurant Management System (Java)

## Overview
This project is an advanced Java-based Restaurant Management System developed as part of coursework. It demonstrates modern Java concepts and real-world application design.

## Features
- Object-Oriented Programming (OOP)
- Stream API and Functional Programming
- Concurrency using ExecutorService
- File Handling using Java NIO
- Logging using Java Logging API
- UML Diagram for system design

## Technologies Used
- Java 21
- Collections Framework
- Stream API
- NIO File Handling

## Project Structure
- Restaurant.java
- Order.java
- Customer.java
- Employee.java (Chef, Waiter, Manager)
- Payment System
- UML Diagram (HTML)
